Machine Learning 시대가 오는가

2016-07-30

머신러닝(machine Learning)의 시대가 오는가? 라는 질문에 대해서 "아니다 또는 모르겠다"라고 말하는 사람은 이 쪽 분야가 아닌 다른 분야나 아예 이 쪽에 관심이 없는 사람일 것이다. 우리가 찾아보지 않더라도 각종 언론에서 몇 일이 멀다하고 기사를 쏟아내고 있으며 실제로 거대 IT 기업들의 머신러닝을 이용한 공개 API나 공개서비스들을 보면 대단하다라는 말이 자동으로 튀어나온다.
그동안은 실생활에 먼 학문연구적 내용이나 Software의 Core 안 쪽에서만 활용되어 '이' 것의 능력을 모르고 있었던 것이다. 이미 '이' 것을 활용한 기술들을 살펴보면 보통 사람들도 놀라움을 표현할 수 밖에 없고 더 나아가 '패러다임의 전환', 'A.I혁명' 등의 단어를 꺼내는 것도 쉽게 볼 수 있다.
(흠.. 진짜 IT랑 관계없는 평범한 분들은 조금(?) 더 설명이 필요하긴 할거같다. ^^;)

머신러닝은 우리의 일상생활에 몰래몰래 깊숙이 파고 들어와 있다. 일기예보, 교통신호, 비행기 스케줄, 온라인 쇼핑, 주식시세/매매/분석 등은 물론이고, 인터넷에서 음악을 듣거나 영화를 보고 사진을 찍어 올리고 SNS에서 친구와 메시지를 주고받을 때도 관여하고 있다. 우리가 보는 뉴스, 할리우드 영화, 소설, 음악조차 머신러닝이 만들어낸다. 사실 이 분야는 아직 초보단계이긴 하다.

예전에 Microsoft사의 인공지능 채팅봇 '테이'가 온라인에 공개된지 24시간이 지나자 인종주의적 차별 발언 등을 쏟아내고 성적인 차별 발언, 편향적 정치적 발언에 욕설도 내뱉었다. 이런걸 보면 아직 기술의 성숙함을 논 할 시기는 아니긴 하지만 '대세' 라는 단어를 붙이기엔 충분한 시절이 아닌가 싶다.
후일담을 보면 테이의 공격적인 차별 언사는 테이가 지닌 '따라하기'(repeat after me) 기능을 악용한 결과로 보이며 테이가 온라인으로 공개된 직후 백인 우월주의자와 여성, 무슬림 혐오자 등이 모이는 익명 인터넷 게시판에 "테이가 차별 발언을 하도록 훈련시키자" 라는 내용의 제안이 올라왔고 이들은 테이에게 "따라해 봐" 라는 말로써 테이의 '따라하기' 기능을 작동시킨 뒤 부적절한 발언을 입력하는 수법을 사용했다고 한다.
일부 사용자들은 테이와 대화를 나누면서 욕설이 섞인 말투와 인종, 성차별 등 극우 성향의 주장을 되풀이해서 테이에 들려주기도 했단다.
그러고보면 역시 인간이 나쁜놈이다. ㅋ

최근에 공개가 계속되는 구글(Google)의 3대 머신러닝 Open API를 살펴보자.

Google Translate API
Google Speech API
Google Cloud vision API
가 있다.

이 API들을 조합해보면 기존에 상상만 하던 것들을 실제로 구현할 수 있게 된다. 아마 스타트업 창업아이템의 갯수가 대폭 늘었을거 같다. 내가 봐도 수없이 많은 아이디어가 떠오르니 말이다. (흠.. 역시 창업을 해야하나.. ^^)
예전에 글쓴이도 스타트업을 준비한적이 있어 이 기술들로 상상했던 아이템들을 현실화하는 생각만으로도 시간이 훌쩍 가는줄 모를정도다. 물론 조금 깊게 들어가보면 여러가지 문제점이 발견되지만, 예전처럼 아예 막막한게 아니라 이젠 손안에 잡힌 도구가 있으니 문제점들을 어떻게 해결할 수 만 있다면 적접적이고 가시적인 결과물을 낼 수도 있게 된 것이다. 위의 언급된 기술들은 사실 최근에 나온것은 아니다. '최근에 공개가' 라는 표현을 쓴 것은 최근에 기술 수준이 급격하게 올라갔기 때문이다.

첫 번째로 구글번역(Google Translate)은 신경망기술(Neural Machine Translation)을 응용해 정확도를 더 높였다. 대상은 영어와 한국어를 포함한 8개 언어로 전세계 언어 중 30%에 해당하며 구글은 최종적으로는 구글 번역이 지원하는 103개 언어 모두에 이 기술을 적용할 계획이란다. 말 그대로 어마어마하다. 스타워즈의 C3PO가 예전엔 미래기술로 보였는데, 지금 구글이 하는 짓(?)을 보니 곧 나올지도 모르겠다.
(하지만 뒷골목 통신에 따르면 스타워즈매니아들은 저딴건 모르겠고 우잉~우잉~ 대는 광선검을 가장 기대하며 기다리고 있다고 한다. ^^?)

글라우드비전(Cloud Vision)은 그 동안 구글 포토등으로 모은 엄청난 이미지 데이터를 바탕으로 기존 이미지 인식을 한차원 더 높여서 나타났다. 사물, 장소, 인물, 문자, 감정까지 판단하여 알려준다. 실제로 대량의 데이터를 적용 시에는 어쩔지 모르겠지만, 활용방도는 무궁무진한거 같다. 실제로 node.js 나 python 등으로 벌써 데모를 해서 보여주시는 분들이 있어 사용의 어려움도 최소화 한거 같다.

자 그럼 실제 이 3대장들을 조합했을 때 어떤일이 일어나는지, 어떤걸 할 수 있는지 머릿속으로 생각만 하지말고 동영상을 보며 느껴보자!!! 아래 동영상은 구글이 Cloud Vision API 베타 버전을 발표하면서 공개한 동영상이다.


머신러닝의 시대는 이미 왔다. 먼저 느끼고 좀 나중에 느끼고의 조금의 차이가 있을 뿐이다. 지인분들은 "이미 이 건 다 아는 사실이야~" 를 외치며 "이제와서 웬 뻘글이냐" 라고 글쓴이에게 돌을 날리고 있지만 "먹고 사느라 바쁨(+5)" 방패로 막으며 살고 있다. 사실 생계걱정이 없다면 이 쪽에 푹빠져 뭔가를 만들어 보고 싶은 생각이 넘쳐흐른다. (첫 번째는 게임을 만들고 싶다. ^^;;)
자~ 이제 머신러닝의 시대가 왔는데 나는 무엇을 할 것인가?? 일단 퇴근부터 하자. ^^
